# Screenshots Guide

## How to Capture Screenshots for README

### Required Screenshots:

1. **demo.png** (1200x600) - Main landing page or template selection
2. **instagram.png** (300x400) - Instagram template result
3. **newspaper.png** (300x400) - Newspaper template result
4. **magazine.png** (300x400) - Magazine template result
5. **comic.png** (300x400) - Comic template result
6. **passport.png** (300x400) - Passport template result
7. **neon.png** (300x400) - Neon template result
8. **stickers.png** (800x400) - Preview page with stickers
9. **customization.png** (800x400) - Text customization page
10. **export.png** (800x400) - Export/download options

### Steps:

1. Open https://thanhnguyxn.github.io/Photo-Booth/
2. Go through the flow for each template
3. Use screenshot tool (Windows: Win + Shift + S)
4. Save to this folder with correct names
5. Update README.md image paths:
   - Replace `https://via.placeholder.com/...` 
   - With `docs/screenshots/filename.png`

### Example:

Replace:
```markdown
![SnapStrip Studio Demo](https://via.placeholder.com/1200x600/667eea/ffffff?text=SnapStrip+Studio+-+Live+Demo)
```

With:
```markdown
![SnapStrip Studio Demo](docs/screenshots/demo.png)
```

### Tips:

- Use high-quality screenshots (at least 1080p)
- Show real photos with stickers/decorations
- Capture diverse templates
- Include UI elements to show features
- Crop out browser chrome for cleaner look
