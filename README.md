## 🎯 Tech Stack

- **HTML5** + Canvas API (photo manipulation, sticker rendering)
- **Tailwind CSS** (responsive UI, gradient effects)
- **Vanilla JavaScript** (no frameworks, lightweight)
- **WebRTC API** (camera access)
- **QRCode.js** (QR code generation)
- **GIF.js** (animated GIF creation)
- **tmpfiles.org API** (temporary file sharing)
- **LocalStorage** (template customization, canvas data)

### Key Algorithms
- 🎯 **Smart Sticker Distribution**: Circular placement around individual photos with collision detection
- 📐 **Dynamic Text Scaling**: Auto-adjusts based on layout and user preference
- 🎨 **Canvas Compositing**: Multi-layer rendering with filters, frames, and overlays

## 🚀 Quick Start

Just visit: **[https://thanhnguyxn.github.io/Photo-Booth/](https://thanhnguyxn.github.io/Photo-Booth/)**

### Basic Flow:
1. 🎯 Click **START**
2. 📐 Choose **Layout** (S, A, B, C, D - single to 6 poses)
3. 🎨 Select **Template** (12 creative options)
4. ✏️ **Customize Text** (for templates like Instagram, Newspaper, etc.)
5. 📷 **Capture Photos** (camera or upload)
   - Choose Auto or Manual capture mode
   - Apply filters in real-time
6. 🎨 **Decorate Preview**
   - Add stickers with drag & drop
   - Change frame colors
   - Hide controls for full preview
7. 📥 **Export & Share**
   - Download as PNG
   - Create animated GIF
   - Share via QR code (1-hour link)

## 🎮 Usage Tips

### Smart Sticker System
- 💡 **Drag & Drop**: Click and drag stickers to reposition
- 🎯 **Auto Placement**: Stickers distribute around individual photos, not center
- 🔄 **Undo**: Made a mistake? Use the undo button (up to 20 steps)
- 🗑️ **Clear All**: Remove all stickers at once
- ⚠️ **Limit**: Max 15 stickers (oldest auto-removed when adding more)

### Capture Modes
- ⚡ **Auto Mode**: Photos taken automatically with countdown timer (3-10 seconds)
- 👆 **Manual Mode**: Click button for each photo when ready

### Text Customization
- 📝 Templates with customization: Instagram, Macbook, Newspaper, Magazine, Movie, Passport, Vintage, Comic, Scrapbook
- 🔤 Adjust text scale with slider (50% - 150%)
- 📱 Text automatically scales for different layouts
